### 문제 설명
규영이는 학교에서 더하기와 빼기를 배웠다.그래서 더하기와 빼기로만 이루어진 수식을 계산하는 계산기를 만들어보기로 했다.(1) 문제 제시입력한 문자열을 수식으로 변환하여 계산 결과를 구하여라.(2) 제약 사항입력된 수식의 최소 길이는 1이고 최대 길이는 1000이다.결과는 32 bit signed integer 범위 안에 있다고 가정해도 좋으며, 계산 과정에서도 overflow는 발생하지 않는다고 가정한다.
### 입력
첫째 줄은 F가 입력된다. F의 길이는 최소 1이고 최대 1000이다.
### 출력
수식의 계산 결과를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
2018+7+27-2000-1-5
```
**예제 출력 1**
```
46
```
### 제약 사항
- 시간 제한   1000ms
- 메모리 제한   256mb

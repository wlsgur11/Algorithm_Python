### 문제 설명
다음 속성을 가지는 딕셔너리를 정의하고 출력해보자.또한 정의된 딕셔너리 변수에 하나의 쌍 {'전화번호':'010-1234-5678'}을 추가하고 출력해봅시다.
### 입력
입력은 없습니다.
### 출력
첫 번째 줄에는 원래 딕셔너리를 출력하고두 번째 줄에는 데이터 쌍을 추가한 딕셔너리를 출력합니다.
### 예제 입력/출력
**예제 입력 1**
```
UNKNOWN
```
**예제 출력 1**
```
{'이름': '신입생', '나이': 20, '전공': '프로그래밍과'}
{'이름': '신입생', '나이': 20, '전공': '프로그래밍과', '전화번호': '010-1234-5678'}
```

### 문제 설명
부산대학교의 통신망은 NNN개의 통신탑과 두 통신탑을 잇는 MMM개의 전선이 존재하는 구간, 전선이 설치되어 있지 않지만 두 통신탑 사이 전선 설치가 가능한 KKK개의 구간으로 이루어져 있다. 끝내 통신탑들을 수리한 산지니는 통신망을 점검하다 통신의 효율화를 위해 통신망을 NNN개의 통신탑이 N−1N - 1N−1개의 전선으로 연결되는 형태로 개편하기로 했다.우리는 통신망의 개편을 위해 아래의 두 가지 작업을 여러 번 할 수 있다.-두 통신탑이 전선으로 이어진 구간의 전선을 제거한다.-전선의 설치가 가능한 두 통신탑 구간에 전선을 추가한다. 전선이 이미 설치된 구간에 전선을 설치할 수 없으며, 전선의 설치가 가능한 구간은 전선이 원래 설치되었다 제거된 구간을 포함한다.구간마다 두 가지 작업을 할 때마다 일정한 비용이 발생하며, 설치가 가능한 구간에 대해서 전선을 제거하는 비용과 설치하는 비용은 동일하다. 설치가 가능 비용의 총합을 최소화하면서 기준을 만족하는 통신망을 만들어보자.NNN개의 통신탑을 N−1N - 1N−1개의 전선으로**연결이 가능한 경우**만 입력으로 주어진다.
### 입력
첫 번째 줄에 통신탑의 개수 NNN, 두 통신탑을 잇는 전선의 개수 MMM, 전선 설치가 가능한 구간의 개수 KKK가 공백으로 구분 지어 주어진다.(1≤N≤105,0≤M,K≤2×105)(1 ≤ N ≤ 10^5, 0 ≤ M, K ≤ 2 \times 10^5)(1≤N≤105,0≤M,K≤2×105)두 번째 줄부터 M+1M +1M+1번째 줄까지 iii번째 전선이 잇는 구간의 두 통신탑의 번호 ui,viu_i, v_iui​,vi​와 전선을 제거하거나 제거 후 다시 설치하는데 필요한 비용 wiw_iwi​가 공백으로 구분 지어 주어진다.(1≤ui,vi≤N,1≤wi≤105,ui≠vi)(1 ≤ u_i, v_i ≤ N, 1 ≤ w_i ≤ 10^5, u_i \neq v_i)(1≤ui​,vi​≤N,1≤wi​≤105,ui​​=vi​)M+2M + 2M+2번째 줄부터 M+K+1M + K + 1M+K+1번째 줄까지 전선을 설치 가능한 iii번째 구간의 두 통신탑 번호 ui,viu_i, v_iui​,vi​와 전선을 설치하거나 설치 후 다시 제거하는 데 필요한 비용 wiw_iwi​가 공백으로 구분 지어 주어진다.(1≤ui,vi≤N,1≤wi≤105,ui\neqvi)(1≤ u_i, v_i ≤ N,1≤ w_i ≤10^5, u_i\neqv_i)(1≤ui​,vi​≤N,1≤wi​≤105,ui​\neqvi​)전선이 설치된 구간과 전선 설치가 가능한 구간은**중복**해서 입력이 들어오지 않으며 설치된 구간은 설치가 가능한 구간의 입력으로 들어오지 않음에 유의하라.
### 출력
통신망을 기준에 맞게 개편할 수 있는 최소 비용을 출력한다. C++, Java의 경우 32비트 정수 범위를 넘을 수 있음에 유의하라.

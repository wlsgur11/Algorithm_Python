### 문제 설명
두 개의 정수를 입력 받고두 정수의 더하기, 빼기, 곱하기, 나누기의 몫과 나머지를 구해봅시다.
### 입력
첫 번째 줄에는 정수 A를,두 번째 줄에는 정수 B를 입력합니다.
### 출력
공백으로 구분하여더하기(A+B), 빼기(A-B), 곱하기(A*B), 몫(A//B), 나머지(A%B)를 출력합니다.
### 예제 입력/출력
**예제 입력 1**
```
50
60
```
**예제 출력 1**
```
110 -10 3000 0 50
```

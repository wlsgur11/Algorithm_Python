### 문제 설명
두 개의 정수를 입력 받아정수 더하기문자열 더하기를 시도해봅시다.
### 입력
첫 번째 줄과 두 번째 줄에 정수를 입력합니다.
### 출력
첫 번째 줄에는 정수 더하기 결과를,두 번째 줄에는 문자열 더하기 결과를 출력합니다.
### 예제 입력/출력
**예제 입력 1**
```
100
99
```
**예제 출력 1**
```
199
10099
```

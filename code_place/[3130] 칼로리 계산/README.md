### 출력
첫 번째 줄은 최소 칼로리두 번째 줄은 최대 칼로리세 번째 줄은 총 칼로리네 번째 줄은 평균 칼로리를 출력합니다.(단, 평균 칼로리는 소수점 둘째 자리까지 출력합니다.)
### 예제 입력/출력


### 문제 설명
값을 입력받아 리스트에 저장하고 리스트를 출력하라.
### 입력
첫 줄에 입력받을 수의 개수를 입력하고, 다음줄부터 한 줄씩 리스트에 넣고자 하는 수를 입력받는다.
### 출력
리스트를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
5
45
46
20
5
9
```
**예제 출력 1**
```
[45, 46, 20, 5, 9]
```

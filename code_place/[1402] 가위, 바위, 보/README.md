### 문제 설명
A라는 사람과 B라는 사람이 가위, 바위, 보를 한다면, 아래 예시처럼(가위→1, 바위→2, 보→3)를 입력 받아서 승,패,무 결과를 출력하는 프로그램을 작성한다.입력이 2 2 인 경우 출력은 "무승부"입력이 1 3 인 경우 출력은 "A승B패"입력이 2 3 인 경우 출력은 "B승A패"입력이 2 4 인 경우 "잘못된 입력"입력이 4 1 인 경우 "잘못된 입력"
### 입력
A와 B의 가위, 바위, 보의 값을 숫자로 입력한다.
### 출력
가위, 바위, 보의 결과를 출력한다.출력결과는 다음 4가지중 하나로 출력된다.A승B패 / B승A패 / 무승부 / 잘못된입력
### 예제 입력/출력
**예제 입력 1**
```
2 2
```
**예제 출력 1**
```
무승부
```

### 문제 설명
산지니는 정보컴퓨터공학부를 위해 몇가지 정책을 내놓았다. NNN명의 정보컴퓨터 공학부 학생은 자신이 원하는 정책에 투표했다.산지니는 과반의 학생이 원하는 정책이 있는지 궁금해졌다. 산지니에게 과반의 학생이 원하는 정책이 뭔지 알려주자.
### 입력
첫번째 줄에 학생의 수 NNN이 주어진다.(1≤N≤1,000,000)(1 \leq N \leq 1,000,000)(1≤N≤1,000,000)두번째 줄에 i번째 학생이 원하는 정책의 번호인 NNN개의 정수 TiT_iTi​가 무작위 순서로 공백으로 구분되어 주어진다.$(1 \leq T_i \leq 1,000,000,000)
### 출력
과반의 학생이 원하는 정책이 있다면 그 정책의 번호를 출력한다.그런 정책이 없다면 −1-1−1을 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
1
361821210
```
**예제 출력 1**
```
361821210
```
**예제 입력 2**
```
2
745874113 264215504
```
**예제 출력 2**
```
-1
```

### 문제 설명
배열 arr에 정수들을 입력한 다음 가장 작은 수를 제거한 배열을 출력하세요.단, 출력하려는 배열이 빈 배열인 경우엔 배열에 -1을 채워 출력하세요.
### 입력
arr은 길이 1 이상인 배열입니다.인덱스 i, j에 대해 i ≠ j이면 arr[i] ≠ arr[j] 입니다.
### 출력
예를들어 arr이 [4,3,2,1]인 경우는 4 3 2를 출력하고, [10]면 -1을 출력 합니다.
### 예제 입력/출력
**예제 입력 1**
```
4 3 2 1
```
**예제 출력 1**
```
4 3 2
```
**예제 입력 2**
```
10
```
**예제 출력 2**
```
-1
```
